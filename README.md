# Newrelic::Rake

NewRelic instrument for rake task.

## Installation

Add this line to your application's Gemfile:

    gem 'newrelic-rake'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install newrelic-rake

## Usage

nothing special to set.
